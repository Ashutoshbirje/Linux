# Command
whoami

---

## Description
Prints the username associated with the current effective user ID.

---

## Syntax
```bash
whoami [options]
```

---

## Options/Flags
- ###  --help : Displays help
    ```bash
    $ whoami --help
    ```
- ### --version : Displays the version
    ```bash
    $ whoami --version
    ```
