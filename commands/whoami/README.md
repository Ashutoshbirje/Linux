# whoami
Print efffective user name
---

## Description
Prints the username associated with the current effective user ID.

---

## Syntax
```bash
whoami [options]
```

---

## Options/Flags
- ###  --help : Display this help and exit
    ```bash
    $ whoami --help
    ```
- ### --version : Display the version and exit
    ```bash
    $ whoami --version
    ```
